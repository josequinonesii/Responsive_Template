Template for Responsive Website Development

This template is to be used as a starting point for developing responsive web designs. This is for front end developers who have a good understanding of how to build a site, and just prefer a starting point. Resize this demo on a computer to see the nav change, or just check it out on your mobile device.

View Project on Github

Features

  • Automatically clones your main navigation and creates a mobile menu.
  • Examples included for several types of responsive elements
  • Hardware Accelerated CSS transitions used instead of jQuery animations to provide silky smooth effects.
  • iOS 6 is tricked into using Hardware Acceleration. (It is disabled by default for some reason.)
  • Multi-Level Dropdowns supported for both main & mobile navigation.
  • Viewport Aware dropdowns that always stay visible.
  • Improved response times due to a script that removes a 300ms standard delay on mobile devices.
  • Includes jQuery, jQuery Easing, jQuery Transit, & more.

Usage

  1. First Download the source files.
  2. If you want to use this layout, go ahead and start styling!
  3. If you prefer to use your own layout, just make sure to keep the wrapper divs for the mobile navigation intact. Keep content within the wrap2 div.
  4. The template will automatically take care of the mobile side of things. Be sure to check out the following examples of how to use other responsive elements.
